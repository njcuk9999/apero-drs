    \section{Some common python 3 features that may be
unfamiliar}\label{some-common-python-3-features-that-may-be-unfamiliar}

Below are a few common features introduced in python 3 that may be
unfamiliar (note they all work in the latest versions of python 3)

    \subsection{The print function}\label{the-print-function}

This is a very basic change. Python 3 will no longer accept the print
statement without parantheses.

For example:

    \begin{Verbatim}[commandchars=\\\{\}]
{\color{incolor}In [{\color{incolor}1}]:} \PY{c+c1}{\PYZsh{} python 2}
        \PY{k}{print} \PY{l+s+s1}{\PYZsq{}}\PY{l+s+s1}{string}\PY{l+s+s1}{\PYZsq{}}
\end{Verbatim}


    \begin{Verbatim}[commandchars=\\\{\}]
string

    \end{Verbatim}

    \begin{Verbatim}[commandchars=\\\{\}]
{\color{incolor}In [{\color{incolor}2}]:} \PY{c+c1}{\PYZsh{} python 3}
\end{Verbatim}


    \begin{Verbatim}[commandchars=\\\{\}]
{\color{incolor}In [{\color{incolor}3}]:} \PY{k}{print}\PY{p}{(}\PY{l+s+s1}{\PYZsq{}}\PY{l+s+s1}{string}\PY{l+s+s1}{\PYZsq{}}\PY{p}{)}
\end{Verbatim}


    \begin{Verbatim}[commandchars=\\\{\}]

    \end{Verbatim}

    \subsection{String formatting}\label{string-formatting}

This is by far the most useful change. In python 2 pushing ints, floats,
strings into another string was done using the \% operator. This has now
been replaced with the use of \{\} and a .format method.

i.e.

    \begin{Verbatim}[commandchars=\\\{\}]
{\color{incolor}In [{\color{incolor}11}]:} \PY{c+c1}{\PYZsh{} python 2}
         \PY{n}{x} \PY{o}{=} \PY{l+m+mf}{3.14159265359}
         \PY{n}{y} \PY{o}{=} \PY{l+m+mi}{100}
