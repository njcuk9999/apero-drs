.. _glossary:

Glossary
========


.. glossary::

  DRS_UCONFIG
      The directory containing the users configurations files (default is :file:`/home/user/apero/`)
      
        
  INSTRUMENT
      This is the instrument used at a specific telescope. Some settings are instrument specific.
      Currently supported instruments are: 
      - `SPIROU`
	
	
	
  ds9
      An astronomical imaging and data visualization application (`see here <http://ds9.si.edu/site/Home.html>`_).
      
      
  pdflatex
      The pdf latex compiler (`see here <https://www.latex-project.org/get/>`_).
