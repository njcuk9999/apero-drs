# Example data folder

This is the example data folder

File structure should be as follows:


- ```/data/calibDB/```

- ```/data/msg/```

- ```/data/raw/```

- ```/data/reduced/```

- ```/data/tmp/```

- ```/data/plot/```

- ```/data/runs/```

- ```/data/telluDB/```

These paths can be changed in ```{DRS_UCONFIG}/{INSTRUMENT}/user_config.ini```
